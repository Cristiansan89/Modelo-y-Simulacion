  • Hipótesis

      Se desea probar la hipótesis:
    • H0: las distintas probabilidades de los sucesos Ai son, P(Ai) = pi ; ∀i = 1; 2; ....; k, donde cada pi es un valor numérico conocido, en el intervalo (0,1).
    • H1: P(Ai) ≠ pi para algún i = 1; 2;...; k

    • Si H0 es verdadero, la variable aleatoria dada por la expresión (1) sigue, para n grande, una distribución X2 con k-1 grados de libertad. Se aceptará H0, si X2 < X2E, donde el segundo miembro es el valor de X2 dado por la tablas para un nivel de significación E y (k-1) grados de libertad.
