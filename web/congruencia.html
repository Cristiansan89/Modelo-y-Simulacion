    Descripción

      Se basan en: Vi+1 = (a Vi + c Vi-k) mod m

      Donde Vi, Vi-k, a, c, k, m son enteros no negativos y arbitrarios que cumplen las siguientes condiciones: Vi >= 0, a >= 0, c >= 0, m > Vi, m > a, m > 0.
