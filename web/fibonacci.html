  • Modelo Fibonnacci

  • Parametros

    • V1 Primer término de la semilla.
    • V2 Segundo término de la semilla.
    • A Constante de control.
    • n Cantidad de números aleatorios que se generan.
    • *Array con n números pseudoaleatorios.

    Descripción

      1. Elegir tres parámetros (enteros no negativos de 3 a 7 dígitos): V1, V2 y A.

      2. V3 = V2 + V1 + k * A, donde k es un parámetro real y entero que se define como:
      0 si V2 + V1 <= A
      -1 en otro caso

      3. Asignar V3 a V2, a V1.

      4. Repetir los pasos del 2 al 3 n veces, siendo n la cantidad de números pseudoaleatorios a generar.
